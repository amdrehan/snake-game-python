# snake-game-python
This is a simple Snake Game built using Python and pygame.

## Why I built this
I am learning Python and wanted to build a small but complete project
to understand game loops, keyboard input, and basic logic.

## Features
- Arrow key movement
- Score tracking
- Game over screen

## Tech Used
- Python
- pygame

## How to run
1. Install pygame:
   pip install pygame
2. Run the game:
   python snake_game.py
